# 剪贴历史

在章鱼速贴里，「剪贴」不是一个动作，而是一份**被记录下来的内容**。

- 系统里的「剪贴板」，是系统当前只保存一份内容的临时空间；
- 章鱼速贴里的「剪贴」，是从剪贴板里**截取下来的一个内容快照**，可以长期保存、搜索和同步。

每当你在系统里复制一段文字、一张图片、一个文件，或是它们的组合时，章鱼速贴都会自动把当下的内容封装成一条「剪贴」，并写入「剪贴历史」。

「剪贴历史」就是这些剪贴的时间轴视图，你可以在这里：

- 回看之前复制过的内容  
- 再次复制 / 粘贴、编辑或删除不需要的剪贴  
- 把有用的剪贴同步到其他设备、或者固定下来长期复用  

我们的设计目标可以用一句话概括：

> **复制即记录，形成可搜索、可同步、可复用的数据流。**

## 剪贴来源是什么？

在章鱼速贴里，「剪贴」只是结果，本质上它们都来自不同的**来源**（Source）。

典型的剪贴来源包括：

- 系统剪贴板：你按下复制 / 剪切时产生的内容  
- 短信内容：从短信中一键提取验证码等信息  
- 屏幕截图：截屏后自动识别并生成可用的剪贴  

章鱼速贴会根据不同来源，自动把这些内容转化为「剪贴」，写入剪贴历史，并标注好来源信息，方便你后续筛选、搜索和回顾。
