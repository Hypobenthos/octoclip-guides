# 剪贴来源：短信

很多时候，收到短信需要先打开手机短信应用并使用，开启之后即可在收到瞬间即可存入剪贴历史并自动同步到其他设备。

## 它会做什么？

当你授权之后，章鱼速贴可以：

- 从新收到的短信中提取特定类型的信息（如验证码、一次性口令等）
- 自动生成一条剪贴，直接出现在剪贴历史或快速输入中
- 让你可以一键粘贴，而不需要先手动选中、复制再切换应用

## 关键信息提取使用

很多时候，你复制短信只是为了拿到其中的一小段信息——比如验证码。
章鱼速贴可以把这些“重复的小动作”自动化。

## 隐私和权限

- 需要你明确授权后才能访问短信内容
- 只会针对特定格式（如验证码短信）进行解析
- 提取到的内容以剪贴形式保存在本地，是否同步完全取决于你的同步设置
