# 剪贴来源：剪贴板

系统剪贴板是章鱼速贴最基础、也是最常用的剪贴来源。

## 工作原理

当你在系统中执行这些操作时：

- 复制（Copy）
- 剪切（Cut）
- 从别的应用中粘贴内容到系统剪贴板

章鱼速贴会在后台**监听系统剪贴板的变化**，一旦检测到新的内容，就会：

1. 读取当前剪贴板里的完整内容（文本、图片、文件、混合内容等）
2. 打包成一条「剪贴」
3. 写入「剪贴历史」，供你之后搜索、同步和复用

简单来说：**你每一次复制，都是在给自己的知识库“打标签”。**

## 为什么要这么做？

- 防止「复制了却没地方找回」
- 记录不止一条，而是形成可以翻页的时间轴
- 为搜索、分类、跨设备同步打基础

如果你只把剪贴板当作「临时中转站」，内容迟早会丢；章鱼速贴则把它变成**可追溯的内容流**。

## 隐私与安全

- 章鱼速贴只在本地读取剪贴板内容
- 是否同步到云端或其他设备，由你在「云同步」「附近同步」等功能中单独开启
- 你可以随时在设置里关闭或者临时暂停剪贴板监听，或为特定应用配置规则开启「排除列表」，自动过滤隐私数据，避免在敏感场景下记录剪贴
