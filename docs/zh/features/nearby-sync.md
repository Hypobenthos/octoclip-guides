# 附近同步

「附近同步」让你在同一网络中的设备之间，**直接**传输剪贴内容，不经过云端，速度更快、延迟更低。

适合这些场景：

- 在电脑上复制，在旁边的手机上直接粘贴  
- 同一局域网内多台设备间频繁互相传内容  
- 不方便使用云同步、或对云存储有顾虑时

## 工作原理（用户向）

当你在一台设备上开启附近同步时，章鱼速贴会：

1. 在局域网内广播自己的存在（如 Wi-Fi / 有线局域网）  
2. 在其他开启附近同步的设备列表中，显示为「可配对设备」  
3. 完成配对后，两台设备之间就建立了一条**点对点传输通道**

之后：

- 当你复制内容时，章鱼速贴会先在本地生成一条「剪贴」  
- 如果你选择发送到某台附近设备，剪贴会通过局域网直接推送过去  
- 对方设备接收后，可以写入剪贴历史，或直接推到系统剪贴板进行粘贴

简单理解：**附近同步 = 同一网络里的“专属空投”，但只为剪贴服务。**

## 使用条件

- 两台设备都安装并登录（或授权）章鱼速贴  
- 都开启了「附近同步」功能  
- 处于同一局域网内（同一 Wi-Fi、同一路由器或同一有线网络）  
- 部分平台可能需要：
  - 允许本地网络访问权限  
  - 通过系统防火墙（首次会有弹窗询问）

## 隐私与安全

- 附近同步的数据**只在局域网内传输**，不经过云端服务器  
- 传输过程会经过加密处理，防止同一网络上的其他设备窃听  
- 每台设备都有自己的标识和昵称，你可以在设置中更改、隐藏或取消配对  
- 不想被附近设备看到时，可以随时关闭附近同步或只允许手动确认的连接请求

## 适合和云同步搭配使用

附近同步更适合**短距离、即时传输**；  
云同步更适合**多设备、跨网络、跨时区的长期协作**。

你可以只开其中一种，也可以两种都开，根据自己的使用习惯来选择。
