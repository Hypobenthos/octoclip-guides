# 云同步

「云同步」让你的剪贴历史、集合、别名和设置，在多台设备之间保持大致一致。  
只要能连到互联网，你就可以在任何一台设备上继续之前的工作。

## 云同步能做什么？

开启云同步后，章鱼速贴可以：

- 在多台设备之间同步：
  - 剪贴历史（按你的设置选择是否同步全部或部分）
  - 收藏集合、别名、标签等结构化信息
  - 部分应用内设置和偏好
- 在新设备上快速恢复你的使用环境，减少重新配置的时间
- 为跨平台使用（如 Windows + macOS + 手机）提供统一的体验

一句话：**把你零散在各个设备上的剪贴，串成一条连续可搜索的时间线。**

## 基础架构（用户向）

为了兼顾可靠性和隐私，云同步通常包含两部分：

1. **云盘存储**  
   - 用于保存你的剪贴数据和结构信息  
   - 可以基于 WebDAV、S3 等云盘 / 对象存储服务实现  
2. **即时同步通道（可选）**  
   - 用于在设备在线时，尽量做到“秒级”同步  
   - 例如，当一台设备复制了一条新的剪贴，其他在线设备可以几乎立刻收到更新

在设置里，你可以看到：

- 连接 / 绑定云盘或云存储服务  
- 选择要同步的内容范围（例如：只同步「集合和别名」）  
- 查看同步状态、错误信息和最近一次同步时间

## 初次配置

1. 打开「云同步」设置页  
2. 选择或登录你要使用的云存储方式  
3. 按提示完成授权  
4. 选择同步范围（默认通常会推荐较安全的范围，如：**只同步集合和别名**）  
5. 在其他设备上重复上述步骤，使用同一账号 / 同一配置文件接入

完成后，设备之间会自动开始数据对齐。根据网络情况和数据量的不同，初次同步可能需要一定时间。

## 同步策略与限制（概述）

- 设备间不是“实时镜像”，而是**尽力保持一致**  
- 离线操作会先保存在本地，等到重新联网后再尝试同步  
- 如果你在多台设备上频繁对同一条内容做改动，最终结果可能以较新的更改为准  
- 如果发现内容不符合预期，你可以：
  - 在剪贴历史中找回旧版本  
  - 临时暂停云同步，检查本地数据后再重新开启  

（具体行为会视后续版本优化而调整，以应用内说明为准）

## 隐私与安全

- 云同步**不会默认开启**，必须由你主动在设置中打开  
- 同步的数据会在传输时加密，存储方式取决于你选择的云服务  
- 对于敏感信息，你可以：
  - 关闭对应集合 / 分类的同步  
  - 或干脆关闭剪贴内容的云同步，仅同步结构化配置（集合、别名等）
